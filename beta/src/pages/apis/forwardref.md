title: forwardRef()

<Intro>

`forwardRef()` lets your component receive a [ref](/learn/manipulating-the-dom-with-refs) and forward it to a child component.

```js
const SomeComponent = forwardRef((props, ref) => {
  // ...
});
```

</Intro>
